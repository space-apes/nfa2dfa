This web application converts a given non-deterministic finite acceptor into an equivalent deterministic finite acceptor. You have the option of providing a list of states and their transitions upon consuming a specific input or calculating the equivalent DFA of pre-determined test-cases.
If you choose to input values, the default first state is named 'q0' and transitions from one state to multiple states require the use of commas between each outgoing state.
